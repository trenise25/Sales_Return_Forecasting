# Quick Start Guide

## Installation
```bash
pip install -r requirements.txt
```

## Usage

### Step 1: Generate Data
```bash
python src/data_generation.py
```

### Step 2: Train Models (5-15 minutes)
```bash
python src/train.py
```

### Step 3: Launch Dashboard
```bash
streamlit run dashboard.py
```

## Dashboard URL
http://localhost:8501
